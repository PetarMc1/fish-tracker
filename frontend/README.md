
# CosmosMC Fish Tracker

A web application to track, visualize, and sort your fish and crab catches on the [CosmosMC Minecraft server](https://cosmosmc.org).

## Features

- **Sorting & Filtering**: Sort by count, rarity, or alphabetical order; search for specific fish.  
- **Rarity Summary**: View statistics by fish rarity tier.  
- **Demo Mode**: Test the tracker without an account.  
- **Remember Login**: Cookie-based login with optional "Remember Me".  

## Demo

You can try the demo without logging in: [Demo Page](https://tracker.petarmc.com/demo)


## Installation

1. **Clone the repository**:

```bash
git clone https://github.com/PetarMc1/fish-tracker.git
cd fish-tracker/frontend
```

2. **Install dependencies**:
```bash
pnpm install
```

3. **Run the development server**:
```bash
pnpm dev
```

4. **Open** http://localhost:3000 **in your browser.**

## Technologies

- **Next.js** – React framework for SSR and client-side navigation.  
- **TailwindCSS** – Utility-first CSS framework.  
- **Framer Motion** – Animations for interactive UI.  
- **js-cookie** – Manage cookies for authentication.
